# Documentação Legada

> Documentos técnicos históricos do sistema SIFAP original, cobrindo o período de 1997 a 2012.

## Conteúdo

| Arquivo                          | Ano  | Descrição                             |
| -------------------------------- | ---- | ------------------------------------- |
| `ARQUITETURA-ORIGINAL-1997.md`   | 1997 | Documento original de arquitetura do sistema |
| `ARQUITETURA-ORIGINAL-1997.docx` | 1997 | Formato original (Word)               |
| `MANUAL-TECNICO-SIFAP-2008.md`   | 2008 | Manual técnico de operações           |
| `MANUAL-TECNICO-SIFAP-2008.docx` | 2008 | Formato original (Word)               |
| `REGRAS-NEGOCIO-2012.md`         | 2012 | Catálogo de regras de negócio         |
| `REGRAS-NEGOCIO-2012.docx`       | 2012 | Formato original (Word)               |

## Observações

- Os arquivos `.md` são versões convertidas para facilitar a leitura; os arquivos `.docx` são o formato original.
- Estes documentos são **material de referência somente leitura** para os times do workshop.
- As regras de negócio documentadas aqui devem ser cruzadas com os programas Natural.
